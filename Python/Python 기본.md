### 배열  
```
a = [] -> 기본선언
b = [100, 200, 300]
c = ['apple', 'orange', 'banana', 'pear']
d = [1, 2, ['black', 'white']]

example = [1, 3, 5, 7, 9]
example[2:3] = [2, 4, 6] -> 2이상 3미만 인덱스에 해당하는 값을 변경  
del example[2:3]이랑 같은 기능!

```

### Tuple  
[]가 아니라 ()로 묶어준다.  
() 생략 가능하다.  
값을 바꿀 수 없다.  
```
tp1 = ()
tp2 = (1,) -> 한 개의 요소만 있을 경우 ,를 꼭 붙여줘야 한다.  
tp3 = (1, 2, 3, 4, 5)
tp4 = (1, 2, (3, 4, 5))
tp5 = 1, 2, 3
```
