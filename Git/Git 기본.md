# Git 

### git 명령어

- commit
- branch
- checkout
- cherry-pick
- reset
- revert
- rebase
- merge


### 1. Git 기본

> git 주요 명령어

#### 1-1 Git 커밋 소개

```bash
$ git commit -m "내용"
```

#### 1-2 Git에서 브랜치 쓰기 및 이동

브랜치: 하나의 커밋과 그 부모 커밋들을 포함하는 작업 내역

브랜치명 bugFix

```bash
$ git branch [브랜치명]
$ git checkout [브랜치명]
```
