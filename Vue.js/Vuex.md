# Vuex    

### 정의  
> vue.js 애플리케이션을 위한 상태 관리 라이브러리.  
> 모든 컴포넌트들에서 접근 가능한 <b>중앙 집중식 데이터 저장소</b>.

### 장점
> MVC 패턴에서 발생하는 구조적 오류 해결.  
> 컴포넌트 간 명시적인 데이터 전달 가능.  
> 여러 컴포넌트에서 같은 데이터 업데이트.  

### 특징  
> state, mutation, actions, getters
